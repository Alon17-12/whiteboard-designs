כל פיצ'ר חדש = React Component
Next.js App Router, Server Components כברירת מחדל. Client Components רק כשצריך אינטראקטיביות.
TypeScript Strict — בלי any
כל interface מוגדר, כל prop typed, כל API response מאומת. strict: true ב-tsconfig.
מדיה דרך Bunny בלבד
וידאו, תמונות וקבצים עוברים דרך Bunny CDN. אין העלאה ישירה לשרת. Bunny Stream לוידאו.
משימות רקע = Trigger.dev
שליחת מיילים, עיבוד וידאו, דוחות, תזכורות — הכל דרך Trigger.dev jobs. לא setTimeout, לא cron ידני.
