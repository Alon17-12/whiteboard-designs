סקירה כללית

מהו WHITEBOARD Studio?

WHITEBOARD Studio הוא מודול הקלטת וידאו ועריכה בסיסית המוטמע ישירות בתוך פלטפורמת ה-LMS של WHITEBOARD. במקום שמרצים יעלו סרטונים מוקלטים מראש מכלי חיצוני, הם יכולים להקליט ישירות מתוך הממשק — לצפות בהקלטה, לערוך אותה (חיתוך, לוגו, אינטרו/אאוטרו, כתוביות) ולפרסם אותה כשיעור בקורס.

קונספט ליבה

Record → Review → Edit → Publish — הכל בלי לעזוב את WHITEBOARD.

  • הקלטה בדפדפן — מצלמה, מיקרופון ושיתוף מסך באמצעות MediaRecorder API
  • מערכת טייקים — כל הקלטה מחולקת לטייקים (takes), אפשר למחוק ולהקליט מחדש חלקים
  • עריכה בסיסית — חיתוך, סידור מחדש, לוגו, מעברים, אינטרו/אאוטרו
  • כתוביות אוטומטיות — יצירת SRT באמצעות OpenAI Whisper, עריכה ידנית, צריבה על הווידאו
  • פרסום ישיר — העלאה ל-Bunny Stream ושיוך לשיעור בקורס

נקודות כניסה

ההקלטה מופעלת ממסך הלייאאוט של הקורס (Course Layout) או מתוך תצוגת השיעור עצמה. כפתור "הקלט שיעור" פותח את ה-Recording Lobby.

למה להקליט בתוך הפלטפורמה?
חוויה אחידה — המרצה לא צריך לעבור בין כלים
ניהול מדיה מרכזי — כל ההקלטות נשמרות ב-Bunny Stream עם מטא-דאטה מסודר
עריכה מיידית — לא צריך תוכנת עריכה חיצונית לפעולות בסיסיות
כתוביות אוטומטיות — תמלול AI מובנה, כולל עברית
פרסום ב-click אחד — מההקלטה ישירות לשיעור בקורס

Recording Stack

  • MediaRecorder API — הקלטה בדפדפן (מצלמה + מסך + אודיו)
  • Codec Strategy — video/webm;codecs=vp8,opus לתאימות cross-browser, Safari fallback ל-MP4/H.264
  • מערכת טייקים — כל טייק = Blob נפרד, נשמר ב-IndexedDB עד להעלאה
  • Pause/Resume — באמצעות MediaRecorder.pause() / resume()
  • timeslice — פרמטר 1000ms ל-flush של chunks, מניעת גלישת זיכרון
Safari Compatibility: Safari לא תומך ב-WebM. יש לזהות את הדפדפן ולהשתמש ב-video/mp4;codecs=avc1 כ-fallback. בדיקה באמצעות MediaRecorder.isTypeSupported().

Upload Stack

  • Uppy.js + @uppy/tus — העלאה resumable ישירות מהדפדפן ל-Bunny Stream
  • תהליך: השרת יוצר video ב-Bunny API → מייצר SHA256 signature → הלקוח מעלה ב-TUS ל-https://video.bunnycdn.com/tusupload
  • Webhook: video.encoding.success מפעיל Trigger.dev job לעדכון השיעור

Editing Stack

  • Client-side: FFmpeg.wasm לפעולות קלות (trim, cut, preview)
  • Server-side: FFmpeg דרך Trigger.dev לפעולות כבדות (transitions, logo burn, subtitle burn, intro/outro concatenation)
  • UI: @designcombo/react-video-editor (MIT) כבסיס לעורך Timeline
  • Templates: Remotion ליצירת אינטרו/אאוטרו פרוגרמטית

Subtitle Stack

  • subtitle.js — פירוק SRT בדפדפן
  • srt-webvtt — המרת SRT ל-VTT ל-HTML5 playback
  • OpenAI Whisper API — /v1/audio/transcriptions עם response_format=srt ליצירה אוטומטית
  • עריכה — ממשק עריכת כתוביות עם התאמת timestamps
  • עתידי: מחולל כתוביות פנימי (Whisper self-hosted)

תהליך ההקלטה

שלבי ההקלטה

  1. הפעלת הקלטה — המרצה לוחץ "הקלט שיעור" ממסך הלייאאוט או מתצוגת השיעור
  2. בקשת הרשאות — מצלמה + מיקרופון (+ שיתוף מסך אופציונלי)
  3. Pre-Recording Lobby — תצוגה מקדימה של המצלמה, בחירת מיקרופון/מצלמה, בחירת מצב הקלטה (מצלמה בלבד / מסך+מצלמה / מסך בלבד). טעינת נכסי שיעור (מצגות HTML, PDF) ותסריט פרומפטר.
  4. ספירה לאחור — 3 שניות countdown לפני כל טייק (גם ראשון וגם resume), עם אנימציה ויזואלית
  5. תחילת הקלטה (TikTok-style) — לחיצה על Record מפעילה הקלטה, לחיצה שנייה = Pause, לחיצה שלישית = Resume (עם countdown). הפרומפטר מתחיל לגלול אוטומטית עם ההקלטה.
  6. במהלך ההקלטה — כפתור Pause/Resume, כפתור Take marker (יוצר גבול סגמנט חדש). אפשרות להחליף מצב (camera/screen+cam/screen) ללא הפסקת ההקלטה.
  7. הערות טייק — הוספת תווית קצרה לכל טייק (מקסימום 30 תווים) לזיהוי מהיר
  8. הכנסת טייק — אפשרות להכניס טייק חדש בין טייקים קיימים (insert between)
  9. מחיקת טייק אחרון — מסיר את הסגמנט האחרון, ממשיך מנקודת הסיום הקודמת
  10. כפתור Undo — ביטול הפעולה האחרונה (מחיקת טייק, סימון גבול, שינוי מצב)
  11. סיום הקלטה — לחיצה על Stop → מעבר למסך Review
  12. Review — Take Gallery — גלריית טייקים ברשת 3 עמודות, עם scrubber וידאו לכל טייק. אפשרות להקליט מחדש טייקים ספציפיים.
  13. Upload — העלאה ל-Bunny Stream ב-TUS עם progress bar (דרך Uppy)
  14. לאחר העלאה — אפשרות "פרסם כמו שזה" או "פתח בעורך"

מצלמה PiP (Picture-in-Picture)

במצב מסך+מצלמה, המצלמה מוצגת כשכבת PiP שניתנת לגרירה על גבי המסך המשותף.

  • צורה: מלבן (rectangle) או עיגול (circle) — ניתן להחלפה
  • מיקום: draggable לכל מיקום על המסך, עם snap-to-corners
  • גודל: slider לשינוי גודל ה-PiP (20%-40% מרוחב המסך)
  • שמירה: מיקום וגודל נשמרים ב-state ומועברים ל-render הסופי

נכסי שיעור (Lesson Assets)

המרצה יכול לטעון חומרי עזר שיוצגו במהלך ההקלטה בתוך iframe:

  • מצגות HTML — תצוגת שקפים עם ניווט קדימה/אחורה
  • קבצי PDF — תצוגה עם דפדוף עמודים
  • כתובת URL — טעינת אתר חיצוני (whitelist בלבד)
  • אינטגרציה: הנכס מוצג כ-layer בהקלטת המסך, או כ-split view לצד המצלמה

אינטגרציית פרומפטר

הפרומפטר מסתנכרן אוטומטית עם מצב ההקלטה:

  • Record = Scroll — כשההקלטה פעילה, הפרומפטר גולל אוטומטית
  • Pause = Stop — עצירת הקלטה עוצרת את הגלילה
  • Resume = Resume Scroll — חידוש הקלטה מחדש את הגלילה מהמיקום הנוכחי
  • Cue Markers — סימני --- בתסריט מאפשרים קפיצה מהירה לנקודות בטקסט

קוד: אתחול MediaRecorder

const startRecording = async (mode: RecordingMode) => {
  const streams: MediaStream[] = [];

  // Camera + Mic
  if (mode !== 'screen-only') {
    const cam = await navigator.mediaDevices.getUserMedia({
      video: { deviceId: selectedCamera, width: 1920, height: 1080 },
      audio: { deviceId: selectedMic, echoCancellation: true },
    });
    streams.push(cam);
  }

  // Screen Share
  if (mode !== 'camera-only') {
    const screen = await navigator.mediaDevices.getDisplayMedia({
      video: { width: 1920, height: 1080 },
      audio: true,
    });
    streams.push(screen);
  }

  // Merge streams
  const combined = mergeStreams(streams);

  // Codec selection
  const mimeType = MediaRecorder.isTypeSupported('video/webm;codecs=vp8,opus')
    ? 'video/webm;codecs=vp8,opus'
    : 'video/mp4';

  const recorder = new MediaRecorder(combined, {
    mimeType,
    videoBitsPerSecond: 2_500_000,
  });

  // Flush chunks every 1s to avoid memory issues
  recorder.start(1000);

  recorder.ondataavailable = (e) => {
    if (e.data.size > 0) {
      currentChunks.push(e.data);
    }
  };

  return recorder;
};

קוד: ניהול טייקים ב-IndexedDB

import { openDB } from 'idb';

const db = await openDB('whiteboard-studio', 1, {
  upgrade(db) {
    db.createObjectStore('takes', { keyPath: 'id' });
  },
});

const saveTake = async (
  recordingId: string,
  orderIndex: number,
  blob: Blob
) => {
  const id = crypto.randomUUID();
  await db.put('takes', {
    id,
    recordingId,
    orderIndex,
    blob,
    createdAt: new Date(),
  });
  return id;
};

const deleteLastTake = async (recordingId: string) => {
  const all = await db.getAllFromIndex('takes', 'recordingId', recordingId);
  if (all.length === 0) return;
  const last = all.sort((a, b) => b.orderIndex - a.orderIndex)[0];
  await db.delete('takes', last.id);
};

קוד: אתחול FFmpeg.wasm

import { FFmpeg } from '@ffmpeg/ffmpeg';
import { fetchFile } from '@ffmpeg/util';

const ffmpeg = new FFmpeg();
await ffmpeg.load({
  coreURL: '/ffmpeg/ffmpeg-core.js',
  wasmURL: '/ffmpeg/ffmpeg-core.wasm',
});

// Trim preview: extract segment from startTime to endTime
const trimPreview = async (
  file: File,
  startTime: number,
  endTime: number
): Promise<Blob> => {
  ffmpeg.writeFile('input.webm', await fetchFile(file));
  await ffmpeg.exec([
    '-i', 'input.webm',
    '-ss', String(startTime),
    '-to', String(endTime),
    '-c', 'copy',
    'output.webm',
  ]);
  const data = await ffmpeg.readFile('output.webm');
  return new Blob([data], { type: 'video/webm' });
};

קוד: המרת SRT ל-VTT לתצוגה מקדימה

import srtToVtt from 'srt-webvtt';

const loadSubtitles = async (srtContent: string) => {
  const srtBlob = new Blob([srtContent], { type: 'text/plain' });
  const vttUrl = await srtToVtt(srtBlob);

  // Add to video element as track
  const track = document.createElement('track');
  track.kind = 'subtitles';
  track.src = vttUrl;
  track.srclang = 'he';
  track.label = 'עברית';
  track.default = true;
  videoElement.appendChild(track);
};

פרומפטר (Teleprompter)

סקירה

מודול פרומפטר מובנה המאפשר למרצה לקרוא תסריט מוכן מראש במהלך ההקלטה. הפרומפטר מסתנכרן אוטומטית עם מצב ההקלטה — כשמקליטים הטקסט גולל, כשעוצרים הגלילה נעצרת.

עורך תסריט

  • טקסט חופשי — עורך טקסט פשוט לכתיבת התסריט
  • מונה מילים — ספירת מילים בזמן אמת
  • הערכת זמן קריאה — חישוב אוטומטי לפי מהירות ממוצעת (150 מילים/דקה עברית)
  • Cue Markers — מפרידים --- (שלושה מקפים) ליצירת נקודות ציון בתסריט לקפיצה מהירה

מצבי תצוגה

Overlay על המסך
שכבה שקופה למחצה מעל תצוגת המצלמה. המרצה רואה את הטקסט מבלי לעזוב את מבטו מהמצלמה.

// Remotion composition for title rendering
import { AbsoluteFill, useCurrentFrame, interpolate } from 'remotion';

export const TitleComposition = ({
  text, subtitle, style, duration, animSpeed
}: TitleProps) => {
  const frame = useCurrentFrame();
  const fps = 30;
  const totalFrames = duration * fps;

  // Fade in over first 15 frames
  const opacity = interpolate(
    frame,
    [0, 15, totalFrames - 15, totalFrames],
    [0, 1, 1, 0],
    { extrapolateRight: 'clamp' }
  );

  return (
    <AbsoluteFill style={{ opacity }}>
      <TitleStyle variant={style} text={text} subtitle={subtitle} />
    </AbsoluteFill>
  );
};

detect-silence

trigger/detect-silence.ts

import { task } from '@trigger.dev/sdk/v3';
import { execSync } from 'child_process';

export const detectSilence = task({
  id: 'detect-silence',
  async run(payload: {
    recordingId: string;
    dbThreshold: number;  // e.g. -30 dB
    minDuration: number;  // e.g. 1.0 seconds
  }) {
    const recording = await prisma.recording.findUniqueOrThrow({
      where: { id: payload.recordingId },
    });

    // 1. Download video from Bunny
    const videoPath = await downloadFromBunny(recording.bunnyVideoId);

    // 2. Use FFmpeg silencedetect filter to find silent segments
    const output = execSync(
      `ffmpeg -i \${videoPath} -af silencedetect=noise=\${payload.dbThreshold}dB:d=\${payload.minDuration} -f null - 2>&1`,
      { encoding: 'utf-8' }
    );

    // 3. Parse silence timestamps from FFmpeg output
    const silences: { start: number; end: number }[] = [];
    const startRegex = /silence_start: ([\d.]+)/g;
    const endRegex = /silence_end: ([\d.]+)/g;

    let startMatch, endMatch;
    while ((startMatch = startRegex.exec(output)) &&
           (endMatch = endRegex.exec(output))) {
      silences.push({
        start: parseFloat(startMatch[1]),
        end: parseFloat(endMatch[1]),
      });
    }

    // 4. Return timestamps for UI to display on timeline
    return { silences, totalFound: silences.length };
  },
});

// API Route: POST /api/studio/create-upload
import crypto from 'crypto';

export async function POST(req: Request) {
  const { lessonId, title } = await req.json();

  // 1. Create video in Bunny Stream
  const bunnyRes = await fetch(
    `https://video.bunnycdn.com/library/\${LIBRARY_ID}/videos`,
    {
      method: 'POST',
      headers: {
        AccessKey: process.env.BUNNY_API_KEY!,
        'Content-Type': 'application/json',
      },
      body: JSON.stringify({ title }),
    }
  );
  const { guid: videoId } = await bunnyRes.json();

  // 2. Generate SHA256 signature
  const expirationTime = Math.floor(Date.now() / 1000) + 3600;
  const signature = crypto
    .createHash('sha256')
    .update(`\${LIBRARY_ID}\${process.env.BUNNY_API_KEY}\${expirationTime}\${videoId}`)
    .digest('hex');

  // 3. Save recording to DB
  await prisma.recording.create({
    data: {
      lessonId,
      teacherId: session.user.id,
      bunnyVideoId: videoId,
      status: 'RECORDING',
    },
  });

  return Response.json({
    videoId,
    signature,
    expirationTime: String(expirationTime),
    libraryId: LIBRARY_ID,
  });
}

export async function POST(req: Request) {
  const payload = await req.json();

  if (payload.EventType === 'video.encoding.success') {
    const recording = await prisma.recording.findFirst({
      where: { bunnyVideoId: payload.VideoGuid },
    });

    if (recording) {
      // Trigger background job
      await tasks.trigger('process-recording-upload', {
        recordingId: recording.id,
        bunnyVideoId: payload.VideoGuid,
        duration: payload.VideoLength,
      });
    }
  }

  return new Response('OK', { status: 200 });
}
